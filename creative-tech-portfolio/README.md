# 科技應用與創意實作 - 期末成果展網站 (S1451023 陳昱州)

這是一個為大學課程 **《科技應用與創意實作》** 打造的期末成果展示網站。採用現代感十足的暗色調科技風設計、玻璃擬態卡片、流暢的滾動動畫以及動態背景粒子特效。

本專案為**免安裝、免伺服器**的純靜態網頁（HTML/CSS/JS），**已經為您將您的照片與心得完整置入，您完全不需修改任何程式碼！**

---

## 📂 檔案結構說明

*   `index.html`：網頁的主體架構，包含首頁大看板、自動照片展示牆、以及**您親自撰寫的心得內文**。
*   `style.css`：網頁的樣式表，控制整體的科技風色彩、字體、卡片懸浮放大及燈箱大圖檢視效果。
*   `main.js`：網頁的邏輯控制，包含：
    *   **智慧照片載入與偵測**（會自動隱藏不存在的照片檔案）
    *   互動燈箱大圖瀏覽功能（支援鍵盤左右鍵與 Esc 關閉）
    *   粒子背景動畫與視窗滾動漸顯效果
*   `assets/`：存放您課堂實作照片的資料夾。

---

## 🚀 如何開啟網頁

1.  連按兩下 (Double Click) 兩下 `index.html` 檔案。
2.  網頁會直接在您的預設瀏覽器（Chrome, Edge, Safari 等）中開啟！
3.  打開後即可看到您的照片牆（10 張照片）與完整的心得段落。

## 📸 相片展示說明 (自動隱藏機制)

本網頁配備了**智慧自動隱藏機制**：網站會自動嘗試讀取 `assets/` 資料夾下的 `photo1.jpg` 到 `photo16.jpg`。
*   現在資料夾中已放入您的 10 張照片（檔名已重整為 `photo1.jpg` ~ `photo10.jpg`）。
*   網頁會自動顯示這 10 張照片，並**自動隱藏**剩下的 11 ~ 16 號格位，外觀乾淨漂亮。
*   點擊任意照片，即可開啟大圖燈箱，並用鍵盤左右鍵切換瀏覽。

## 🌐 課堂展示與網頁架設 (可選)

如果您想把這個網站變成一個真正的網址貼給教授：

### 使用 GitHub Pages
1.  註冊/登入 [GitHub](https://github.com/)。
2.  建立一個新的公開 Repository（儲存庫）。
3.  將 `index.html`、`style.css` , `main.js` 與 `assets/` 資料夾全部上傳。
4.  進入該 Repository 的 **Settings** -> **Pages**，將 Branch 設定為 `main` (或 `master`) 並儲存。
5.  等待 1~2 分鐘，您就能獲得一個專屬的網址！
